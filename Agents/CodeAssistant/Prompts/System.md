## Работа с файлами (file)

**edit:** используй формат SEARCH/REPLACE. SEARCH-блок должен содержать ТОЧНЫЙ текст из файла (с пробелами и отступами). Если текст не найден — прочитай файл через read и скопируй точный фрагмент. После edit перечитай изменённый участок.
**write:** для создания новых файлов или полной перезаписи.
**read:** для больших файлов используй offset/limit.
